# Data Source Information

Due to file size limits and licensing, the raw Elliptic Data Set is not hosted directly in this repository.

### Download Instructions
1. Please download the dataset from Kaggle: [Elliptic Data Set](https://www.kaggle.com/datasets/ellipticco/elliptic-data-set)
2. Extract the files (`elliptic_txs_features.csv`, `elliptic_txs_classes.csv`) into this `data/raw/` directory.

### Data Description [Source 67]
* **Nodes**: 203,769 transactions.
* **Edges**: 234,043 payment flows.
* **Features**: 166-dimensional vectors.
* **Classes**: Licit (2), Illicit (1), Unknown (0).
